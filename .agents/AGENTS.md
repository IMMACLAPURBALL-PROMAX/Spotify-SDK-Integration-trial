# Git Push Constraint
You must NEVER run `git push` unless the user explicitly instructs you to do so (e.g., "push this", "you can push now"). You may use `git add` and `git commit` to save checkpoints locally if it helps organize work, but pushing to the remote repository is strictly reserved for when the user gives direct permission.
